# ⬡ Job Tracker

A clean, minimal job application tracker built with React. Track your pipeline, interview stages, salary targets, and follow-up dates — all in one place, saved locally to your browser.

**[→ Live Demo](https://job-tracker-vika29-gif.vercel.app)** <!-- Update with your actual Vercel URL after deploying -->

---

## What it does

Keeping track of job applications in a spreadsheet is frustrating. This replaces that with something that actually feels good to use.

- **Pipeline at a glance** — see how many applications are at each stage
- **Interview timeline** — log every phone screen, technical, panel, and offer with dates and notes
- **Stats** — response rate, active count, offers, interview rate
- **Filter & search** — by status, stage, or company name
- **Persistent** — saves to your browser's local storage, no account needed

## Tech stack

- [React 18](https://react.dev/)
- [Vite](https://vitejs.dev/)
- No backend, no database — pure client-side

## Running locally

```bash
# Clone the repo
git clone https://github.com/vika29-gif/job-tracker.git
cd job-tracker

# Install dependencies
npm install

# Start dev server
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).

## Deploy your own

This deploys to [Vercel](https://vercel.com) in about 2 minutes:

1. Fork this repo
2. Go to [vercel.com](https://vercel.com) → New Project → Import your fork
3. Click Deploy — no configuration needed
